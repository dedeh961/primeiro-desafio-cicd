# Instruções

## Rodar o projeto localmente

``` bash
make dev
```

``` bash
kubectl logs my-ingress-ingress-nginx-controller-6b9756d5ff-sxftp
```

docker pull public.ecr.aws/j8d2p8u7/desafionext/api-flask:ecb4c64
